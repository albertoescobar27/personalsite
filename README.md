# Sitio Web Personal - Alberto Escobar

Este proyecto es un sitio web personal que funciona tarjeta de presentación digital para **Alberto Escobar**, Ingeniero en Tecnologías Computacionales.

## 🔗 Ver en vivo
Puedes visitar el sitio web en la siguiente dirección:
[https://albertoescobar27.github.io/personalsite/](https://albertoescobar27.github.io/personalsite/)

## 🚀 Sobre el Proyecto
El sitio muestra información sobre el perfil profesional, habilidades técnicas y formas de contacto de Alberto.

### Secciones principales:
- **Sobre mí:** Breve descripción del perfil profesional y experiencia.
- **Habilidades:**
  - **Desarrollo Web:** HTML5, CSS3, JavaScript (ES6+), React.js, integraciones con Agentes de IA.
  - **Backend y Desarrollo de Software:** Ecosistema .NET (C#, Visual Basic).
  - **Bases de Datos:** MS SQL Server, PostgreSQL, Supabase.
- **Contacto:** Enlaces para correo electrónico, WhatsApp, perfiles de LinkedIn y GitHub, y la descarga del CV en PDF.

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura del sitio.
- **CSS3:** Diseño y estilos personalizados.
- **Google Fonts:** Tipografías modernas (Merriweather, Montserrat, Sacramento).

## 📂 Estructura del Proyecto
- `index.html`: Archivo principal del sitio.
- `assets/`: Carpeta contenedora de todos los recursos estáticos.
  - `css/`: Hojas de estilo del sitio.
  - `images/`: Recursos visuales e ilustraciones.
  - `docs/`: Documentos descargables (como el CV en PDF).
  - `icons/`: Favicon y elementos de identidad visual.
